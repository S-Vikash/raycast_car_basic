# Raycast Car Controller (Godot 4)

A simple 3D raycast-based car controller built in Godot 4.

## Features

* Raycast suspension system
* Vehicle acceleration
* Braking
* Reverse movement
* Steering controls
* Spring Arm camera for smooth third-person follow
* Lightweight and beginner-friendly implementation

## Controls

| Key | Action          |
| --- | --------------- |
| W   | Accelerate      |
| S   | Brake / Reverse |
| A   | Steer Left      |
| D   | Steer Right     |

## Overview

This project demonstrates how to create a basic vehicle controller without using Godot's built-in vehicle nodes. The car uses raycasts to simulate wheel suspension and maintain ground contact, providing a foundation for learning vehicle physics and custom controller development.

## Built With

* Godot 4
* GDScript
